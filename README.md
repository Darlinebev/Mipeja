Static hotel website for Mipeja Hotel in Busia, Kenya. Built with Astro and Tailwind CSS, optimized for fast load times and mobile-first browsing. Booking redirects to a trusted external partner.

## Features

- Responsive, modern landing page
- Room highlights with Kenyan Shillings pricing
- Amenities, safety, policies, and guest reviews
- WhatsApp chat and social media links
- Virtual tour CTA and location map embed

## Quick start

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Content updates

- Update hotel content and links in the data file.
- Replace SVG placeholders in the public images folder with real photography when available.

## Booking link

The booking button redirects to the configured external booking URL. Update it in the data file.
